# SAI AUROSY Mini App — Frontend

React + Vite frontend for the SAI AUROSY Telegram Mini App. Part of the monorepo; run from root with `npm run dev` or `npm run dev:frontend`.

- **Framework:** React 19, Vite 7
- **3D:** Three.js, React Three Fiber, @react-three/drei (Event Mode Demo)
- **Routing:** React Router 7

See [Frontend Architecture](../docs/architecture/frontend-architecture.md) and [Screen Map](../docs/ux/screen-map.md) for structure and navigation.
